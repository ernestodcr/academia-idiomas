# 🏫 Academia de Idiomas - Sistema de Gestión Escolar

¡Bienvenido! Este es un proyecto **Full Stack** desarrollado en Python diseñado para la administración integral de centros educativos y academias de idiomas.

## 🚀 Características Clave
* **Arquitectura Robusta:** Desarrollado bajo el patrón Modelo-Vista-Controlador (MVC).
* **Persistencia de Datos:** Gestión relacional optimizada mediante base de datos.
* **Automatización:** Generación dinámica de reportes en formatos Excel.
* **Analítica Visual:** Gráficos estadísticos interactivos integrados en la interfaz web.

## 🛠️ Tecnologías Utilizadas
* **Backend:** Python, Flask, SQLAlchemy
* **Frontend:** HTML5, CSS3, JavaScript (Plantillas Jinja2)
* **Reportes y Datos:** Pandas, OpenPyXL, Matplotlib
* **Base de Datos:** SQLite / MySQL

## 📦 Cómo Ejecutar el Proyecto Localmente

Si deseas probar la aplicación en tu entorno local, sigue estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/ernestodcr/academia-idiomas.git
   cd academia-idiomas
   ```

2. **Crear y activar un entorno virtual:**
   ```bash
   python -m venv venv
   # En Windows:
   .\venv\Scripts\activate
   ```

3. **Instalar las dependencias:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Iniciar la aplicación:**
   ```bash
   python app.py
   ```
   *Abre tu navegador en `http://127.0.0.1:5000`*
